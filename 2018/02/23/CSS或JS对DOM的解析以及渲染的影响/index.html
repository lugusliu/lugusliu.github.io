CSS或JS对DOM的解析以及渲染的影响

通常网页的 css 样式一般写在 <head> 标签内，而 js 脚本一般放在 <body> 标签的最底部或在外链的脚本上使用 defer 或 async 属性。这些都是为了优化样式或者脚本给 DOM 的解析或渲染所带来的影响。下面就将结合具体的例子做具体的探讨
